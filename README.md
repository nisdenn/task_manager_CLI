# 📝 Task Manager CLI

A simple command-line Task Manager built using Python.  
This project allows users to manage daily tasks directly from the terminal.

## 🚀 Features
- Add new tasks
- View all tasks
- Delete tasks
- Persistent storage using JSON

## 📦 Tech Stack
- Python 3
- JSON for data storage

## ▶️ How to Run

1. Clone this repository:
```bash
git clone https://github.com/your-username/task-manager-cli.git
````

2. Go to project folder:

```bash
cd task-manager-cli
```

3. Run the program:

```bash
python main.py
```

## 📁 File Structure

```
task-manager-cli/
│── main.py
│── tasks.json (auto-generated)
```
